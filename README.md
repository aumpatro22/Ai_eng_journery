## Overview

This repository documents a focused journey into **Artificial Intelligence**, **Machine Learning**, and practical **AI Engineering**.

The goal is simple: study important concepts deeply, implement them from scratch or with industry tools, and build a public body of work that reflects both **technical growth** and **engineering discipline**.

Rather than collecting random notebooks, this repository is being developed as a structured learning lab with clear progression, reusable experiments, and implementation-first thinking.

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/aumpatro22/Ai_eng_journery.git
cd Ai_eng_journery
```

### 2. Install dependencies

```bash
pip install numpy pandas scikit-learn matplotlib jupyter
```

### 3. Open the notebooks

Use either:

- Jupyter Notebook
- Jupyter Lab
- VS Code with the Jupyter extension

### 4. Run the notebook cells step by step

As the repository grows, a `requirements.txt` file will be added for simpler setup.
