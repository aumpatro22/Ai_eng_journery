AI Engineering Learning Lab

This repository documents my daily journey of learning and implementing Machine Learning and AI concepts. Each folder represents a topic where I study the theory and then implement the concept in Python using libraries such as NumPy, Pandas, and Scikit-learn.

The goal of this repository is to build practical AI engineering skills by turning theoretical concepts into working code, experiments, and mini-projects.

Topics Covered:
• Supervised Learning
• Regression Models
• Classification Algorithms
• Optimization Methods
• Deep Learning Foundations
• NLP & Large Language Models

Each implementation includes:
- concept explanation
- practical implementation
- experimentation
- debugging insights

This repository acts as my public AI engineering lab where I continuously learn, build, and improve.
